# HourGlass

HourGlass is a WPF .NET 8 desktop timer with an animated hourglass. It is designed as a
focused, Pomodoro-style timer with playful visuals and a completion ding.

## Features
- Animated hourglass with sand flowing from top to bottom
- Set duration using a minutes slider (1â€“60)
- Play/Pause control with icon button
- Reset button to reset the timer and sand
- Flip button appears when the timer completes to restart with a flip animation
- Always-on-top toggle (default on)
- Completion sound when time runs out

## Run
```bash
dotnet run --project HourGlass/HourGlass.csproj
```
